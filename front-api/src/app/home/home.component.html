TRY OUR API

Please, sign up to use our fast and easy-to-work recipe API.
As a start to use DessertAPI products, we recommend our Recipe API 1.0. For more functionality, please consider our products, which are included in professional collections.

Make one API call and receive essential recipes data in one response:
- Name of the recipe
- Description of the recipe
- A picture of what it is supposed to look like
- Ingredients needed
- Difficulty of the recipe
- Duration of this cooking
- Score from others cookers
- Budget
- Anf of course, different steps to realize the recipe !

- A query is used to retrieve all recipes.
- Another will show you the details of a particular recipe (by clicking on it)
- Clicking on the cross next to the recipe will invoke the recipe deletion request
- Connect using the login form in order to be able to add a new recipe
- There is only one administrator role (requested in order to add or update a recipe)
- Finally, use the form, enter a new recipe to add (the add request will be called)

You can read with Swagger the How it works and enjoy using our powerful recipe API right now.
