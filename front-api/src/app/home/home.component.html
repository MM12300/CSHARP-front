TRY OUR API

Please, sign up to use our fast and easy-to-work recipe API.
To start using DessertAPI functionalities, we recommend our Recipe API 1.0. For more service, please consider our products, which are included in professional collections.

Make an API call and receive expected recipes return data:
A recipe is composed of : - A name
- A description
- A picture of what it is supposed to look like
- A list of all the ingredients needed
- The difficulty of the recipe
- The duration
- The score from the others bakers
- The estimated budget
- And of course, different steps to realize the recipe !

About the API: - A query is used to retrieve all recipes.
- Another query will show you the details of a particular recipe (by clicking on it)
- Clicking on the cross next to the recipe will invoke the recipe deletion request
- Connect using the login form in order to add a new recipe
- There is only one administrator role (requested in order to add or update a recipe)
- Finally, use the form, insert a new recipe informations and steps then press the button 'Create Recipe' to add the new created recipe into your book(the add request will be called)

You can check How it works with Swagger and enjoy using our powerful recipe API right now.
